# ReadySet

1. 저장소 클론
   ```
   git clone
   cd ReadySet
   ```
   
2. 가상환경 생성
   
   ```
   python -m venv venv
   ```

3. 가상환경 활성화
   ```
   source venv/bin/activate  # mac
   venv\Scripts\activate.ps1 # Windows powershell
   ```
   
4. 패키지 설치
   ```
   pip install -r requirements.txt
   ```

5. 프로젝트 실행
   ```
   python -m api.api
   ```
